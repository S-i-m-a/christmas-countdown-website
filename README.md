# Christmas Countdown Website

A responsive Christmas Countdown web application developed using HTML, CSS, and JavaScript.  
The website dynamically displays the remaining time until Christmas with a festive and user-friendly interface.

## Live Demo
https://s-i-m-a.github.io/christmas-countdown-website/

## Project Overview
This project is a static frontend web application designed to demonstrate:
- Dynamic time-based calculations using JavaScript
- Responsive UI design using modern CSS
- Clean and structured HTML layout

It can be used as a mini project, portfolio project, or frontend practice application.

## Features
- Real-time countdown to Christmas
- Automatic updates without page refresh
- Fully responsive design for desktop and mobile devices
- Lightweight and fast-loading static website
- Clean and modern festive UI
